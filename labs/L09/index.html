Лабораторна робота №9

Аналіз можливостей автоматизації тестування.

← Назад до меню

Мета

Навчитися оцінювати доцільність автоматизації тестів на основі критеріїв: стабільність, повторюваність, складність реалізації, витрати на підтримку, бізнес-важливість та очікувана вигода.

Навчальний ресурс

У роботі використовується “Automation Assessment Board” — набір тестів з різних модулів. Студент аналізує кожен тест і визначає: автоматизувати / залишити ручним / частково автоматизувати.

Відкрити Automation Assessment Board (ЛР9)

Що потрібно зробити

  1. Відкрити Board і ознайомитися з критеріями автоматизації.
  2. Проаналізувати список тестів та заповнити рішення для кожного.
  3. Вибрати мінімум 8 тестів як кандидатів на автоматизацію.
  4. Побудувати “Automation Candidate List” з пріоритизацією.
  5. Сформувати висновки та рекомендації.

Що здати

  • Звіт ЛР9
  • Таблиця класифікації тестів (Automate/Manual/Partial)
  • Automation Candidate List (8+ тестів)
  • Скріншоти (мінімум 6)
