Виконання HTTP-запитів до API.

← Назад до меню

Мета

Навчитися виконувати HTTP-запити до API (GET, POST, PUT, DELETE), аналізувати статус-коди та тіло відповіді, фіксувати результати у звіті.

Навчальний сайт для роботи з API

У цій лабораторній використовується “API Console” — вбудований симулятор API. Ви не пишете код, а відправляєте запити через форму, як у Postman.

Відкрити API Console (ЛР6)

Що потрібно зробити

  1. Відкрити API Console і прочитати блок “About / API Specification”.
  2. Виконати мінімум 10 HTTP-запитів (GET/POST/PUT/DELETE).
  3. Для кожного запиту зафіксувати: метод, URL, тіло запиту (якщо є), статус, відповідь.
  4. Зробити скріншоти: мінімум 6.
  5. Оформити звіт і зробити висновок.
