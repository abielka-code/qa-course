Лабораторна робота №8

Аналіз результатів тестування.

← Назад до меню

Мета

Навчитися аналізувати результати тестування на основі тестового прогону: обчислювати показники якості (Pass rate, Fail rate), визначати найбільш проблемні модулі, формувати висновки та коротку тестову звітність.

Навчальний ресурс

У роботі використовується “Test Run Analyzer” — сторінка з готовими результатами тестового прогону. Завдання студента — виконати аналіз, зробити розрахунки і сформувати звіт.

Відкрити Test Run Analyzer (ЛР8)

Що потрібно зробити

  1. Відкрити Analyzer та ознайомитися з даними тестового прогону.
  2. Порахувати загальні показники: кількість тестів, Pass/Fail/Blocked/Skipped.
  3. Порахувати Pass rate та Fail rate у відсотках.
  4. Визначити модулі з найбільшою кількістю Fail.
  5. Зробити висновок про якість релізу і рекомендації щодо регресії.
  6. Оформити підсумковий звіт.
