Лабораторна робота №15

Підсумковий міні-проєкт: QA пакет документації

Опис роботи

Ця лабораторна є підсумковою. Студент виконує роль тестувальника та готує повний набір QA-документації для навчального веб-застосунку.

Що потрібно зробити

  • Ознайомитися з функціоналом target-сайту (Mini Checkout App).
  • Побудувати перелік перевірок для основних сценаріїв.
  • Скласти чек-лист, тест-кейси, баг-репорти.
  • Зібрати докази дефектів через UI + DevTools (Console/Network).
  • Підготувати фінальний Test Summary Report та рішення Go/No-Go.
Відкрити target (L15) Повернутися на головну

Артефакти, які потрібно здати

  • Чек-лист (мінімум 25 пунктів).
  • Тест-кейси (мінімум 10–12).
  • Баг-репорти (мінімум 5 або всі знайдені).
  • Test Summary Report (фінальний).
  • Evidence: скріншоти 10+ (UI + Console + Network).
