QA Course Hub

Навчальний ресурс для виконання лабораторних робіт з тестування ПЗ.

Лабораторні роботи

  • ЛР1 — Ознайомлення з тестовим веб-застосунком
  • ЛР2 — Планування тестування. Складання Test Plan
  • ЛР3 — Розробка чек-листа тестування
  • ЛР4 — Створення тест-кейсів
  • ЛР5 — Виявлення та документування дефектів
  • ЛР6 — Виконання HTTP-запитів до API
  • ЛР7 — Розробка API-сценаріїв тестування
  • ЛР8 — Аналіз результатів тестування
  • ЛР9 — Аналіз можливостей автоматизації
  • ЛР10 — Підсумкове тестування та звіт
  • ЛР11 — Тест-дизайн: граничні значення та еквівалентні класи
  • ЛР12 — Негативне тестування та валідація введених даних
  • ЛР13 — Тестування кросбраузерності та адаптивності
  • ЛР14 — Аналіз логів та збір доказів дефекту (DevTools)
  • ЛР15 — Підсумковий міні-проєкт: QA пакет документації

  • SV12. CI/CD для тестувальника

Правила виконання

  • Студенти не пишуть код. Уся робота — тестування та документація.
  • Кожна лабораторна має окремий навчальний сайт для тестування.
  • Результат здачі: звіт (Google Docs / PDF) з артефактами тестування.
